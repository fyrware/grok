# grok
Git-based package manager for C++

### install
In order to install `grok`, simply run the correct install script for your OS:

- linux: [install_linux.sh](https://github.com/fyrware/grok/blob/master/scripts/install_linux.sh)

### commands

- **make**
- **use <package_name>**
- **update <package_name>**
- **apply**

---

- **sync**
- **help**
